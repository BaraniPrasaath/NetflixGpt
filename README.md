# React + Vite
 - npm create vite@latest
 - then it will install all the files and folders, and clear all the css files and unnecessary files.
 - then install the tailwindcss and configure that in the project.
 - Built,
   - Header
   - Body
   - Login
   - Sign Up
   - Welcome
   - Routing of App
   - Protected route with the AuthLayout
   - Implemented form validation
   - Implemented authentication using Vikky's node.js backend
 - FireBase
   - Steps for installation and Deployment
      1. Install firebase CLI - "npm install -g firebase-tools"
      2. Login to Firebase - firebase login
      3. Initialize Firebase - firebase init, then select hoisting
      4. Deployment - firebase deploy
 - Configured with firebase & Deployed The Application to "https://netflixgpt-fc73a.web.app/";
